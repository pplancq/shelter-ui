# ShelterUI Design System CheckboxInput

The `CheckboxInput` component provides a customizable checkbox input with consistent styling. It is used to allow the user to select one or more options from multiple choices.

## Usage

```tsx
import { CheckboxInput } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return <CheckboxInput name="options" value="option1" checked />;
};
```
