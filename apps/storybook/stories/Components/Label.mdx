# ShelterUI Design System Label

The `Label` component provides an accessible label for form fields, with support for required state indication.

## Usage

```tsx
import { Label } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <Label htmlFor="my-input" required>
      Username
    </Label>
  );
};
```
