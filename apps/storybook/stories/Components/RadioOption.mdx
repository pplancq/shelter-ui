# ShelterUI Design System RadioOption

The `RadioOption` component extends the `RadioInput` component by adding integrated label capabilities, positioning and styling.

## Usage of the RadioOption Component

```tsx
import { RadioOption } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return <RadioOption id="option1" name="choice" value="option1" label="Option 1" labelPosition="right" />;
};
```
