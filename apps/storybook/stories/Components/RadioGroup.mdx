# ShelterUI Design System RadioGroup

The `RadioGroup` component provides a container for radio options, allowing users to select one option from a group. It includes proper accessibility features, labeling, error states, and helper text support.

## Usage

```tsx
import { RadioGroup, RadioOption } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <RadioGroup label="Choose an option" name="my-radio-group" layout="stacked" itemsLayout="inline">
      <RadioOption label="Option 1" value="option1" />
      <RadioOption label="Option 2" value="option2" />
      <RadioOption label="Option 3" value="option3" />
    </RadioGroup>
  );
};
```
