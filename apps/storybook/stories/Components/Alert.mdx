# ShelterUI Design System Alert

The `Alert` component is a versatile UI element used to display important messages to users. It supports different severity levels, customizable titles, optional messages, and dismissible functionality with proper accessibility features.

## Usage of the Alert Component

```tsx
import { Alert } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <Alert variant="info" title="Information" onClose={() => console.log('Alert closed')}>
      This is an informational message for the user.
    </Alert>
  );
};
```

## Alert Variants

The `Alert` component supports four different variants, each with specific semantic meaning and visual styling:

### Info Alert

Used for general informational messages that don't require immediate action.

### Warning Alert

Used to warn users about potential issues or important information that requires attention.
