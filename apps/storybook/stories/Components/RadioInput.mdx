# ShelterUI Design System RadioInput

The `RadioInput` component provides a customizable radio button with error state management. It is used to allow the user to select a single option from multiple choices.

## Usage

```tsx
import { RadioInput } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return <RadioInput name="option" value="option1" checked isInvalid />;
};
```
