# ShelterUI Design System HelperText

The `HelperText` component provides additional context or feedback for form fields, supporting both informational and error states.

## Usage

```tsx
import { HelperText } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <>
      <HelperText>This is some helpful information.</HelperText>
      <HelperText error>This is an error message.</HelperText>
    </>
  );
};
```
