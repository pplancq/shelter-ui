# ShelterUI Design System CheckboxField

The `CheckboxField` component extends the `CheckboxInput` component by adding label and helper text capabilities, enabling richer form field interactions and validations.

## Usage

```tsx
import { CheckboxField } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <CheckboxField
      label="Accept terms and conditions"
      name="terms"
      value="accepted"
      textHelper="Please read our terms and conditions"
      errorMessage="You must accept the terms to continue"
      required={true}
    />
  );
};
```

## CheckboxField Props

The CheckboxField component inherits all [`<CheckboxInput/>`](?path=/docs/components-checkboxinput--checkboxinput) component props (except `indeterminate`) and extends them with the following props:

    </tr>
    <tr>
      <th>
        <code>errorMessage</code>
      </th>
      <td>
        <code>ReactNode</code>
      </td>
      <td>-</td>
      <td>Error message displayed below the checkbox field. Takes priority over textHelper when both are provided.</td>
    </tr>
  </tbody>
</table>
