# ShelterUI Design System CheckboxGroup

The `CheckboxGroup` component provides a container for multiple checkbox options, allowing users to select one or more options from a group. It includes accessibility features, labeling, error states, and helper text support.

## Usage

```tsx
import { CheckboxGroup, CheckboxField } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <CheckboxGroup label="Choose options" name="my-checkbox-group" layout="stacked" itemsLayout="inline">
      <CheckboxField label="Option 1" value="option1" />
      <CheckboxField label="Option 2" value="option2" />
      <CheckboxField label="Option 3" value="option3" />
    </CheckboxGroup>
  );
};
```
