# ShelterUI Design System Input

The `Input` component provides a customizable text field, with error state management and optional icons or elements before/after the field.

## Usage

```tsx
import { Input } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <Input
      placeholder="Your text here"
      error={false}
      startAdornment={<Icon icon={searchIcon} />}
      endAdornment={<Icon icon={eyeIcon} />}
    />
  );
};
```
