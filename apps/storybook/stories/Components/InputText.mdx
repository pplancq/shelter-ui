# ShelterUI Design System InputField

The `InputField` component extends the `Input` component by adding label and helper text capabilities, enabling richer form field interactions and validations.

## Usage

```tsx
import { InputField } from '@pplancq/shelter-ui-react';

const MyComponent = () => {
  return (
    <InputField
      placeholder="Your text here"
      label="Input Label"
      textHelper="Helper text"
      errorMessage="Error message"
      required={true}
      layout="inline"
    />
  );
};
```
