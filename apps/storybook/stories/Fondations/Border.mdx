# ShelterUI Design System border

The border radius defines the roundness of the corners of the elements and reinforces the visual identity of an interface.

# Description

A border radius brings softness or rigidity to the corners of interface elements.
It is essential for harmonizing visuals while giving a modern, engaging feel.
Values should be consistent across all components to reinforce visual unity and create an aesthetic hierarchy.

# Rules of use

- **Standardization**: Use a progressive scale to adapt to different contexts.
  - 0px: No rounding (clean edges, tables).
  - 4px: Light rounding for classic forms or buttons.
  - 8px: Intermediate rounding for cards and panels.
  - 16px: Strong rounding for elements such as modals and large sections.
  - 50%: Fully rounded corners for circular elements (avatars, badges).
- **Visual consistency**: Avoid using arbitrary or non-standardized values.
- **Context of use**: Adapt the degree of rounding to the role of the component (e.g. 4px for utility elements and 16px for calls to action).
