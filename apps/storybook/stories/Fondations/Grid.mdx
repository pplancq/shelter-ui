# ShelterUI Design System grid

The grids defined in ShelterUI enable content to be organized coherently and fluidly, adapting to different screen sizes.
Grid configuration varies according to device type, to deliver an optimal user experience.

## Usage Component Grid

```tsx
import { Grid } from '@pplancq/shelter-ui';

const MyComponent = () => {
  return (
    <Grid container>
      <Grid colSpan={4}>1</Grid>
      <Grid colSpan={4}>2</Grid>
      <Grid colSpan={4}>3</Grid>
    </Grid>
  );
};
```
