# ShelterUI Design System Icon

The `Icon` component is used to display SVG icons in various sizes and styles.
It supports customization through props like `size`, `isCircle`, and `className`.
By default, the `Icon` component has the `role` attribute set to `presentation`.

## Usage of the Icon Component

```tsx
import { Icon } from '@pplancq/shelter-ui-react';
import homeIcon from '@pplancq/shelter-ui-icon/icon/home.svg';

const MyComponent = () => {
  return <Icon icon={homeIcon} size="large" />;
};
```
