# Getting Started with ShelterUI

Follow these simple steps to integrate ShelterUI into your project:

## Installation

Install the packages via npm:

```bash
npm install @pplancq/shelter-ui-react @pplancq/shelter-ui-icon @pplancq/shelter-ui-css
```

## Import the CSS (Recommended)

To use the predefined styles and theme tokens, import the CSS package in your project:

```tsx
import '@pplancq/shelter-ui-css/css/shelter-ui.css';
```

## Use a Component

Now, use one of the React components in your application:

```tsx
import { Button } from '@pplancq/shelter-ui-react';

export default function App() {
  return <Button>Click Me</Button>;
}
```

## Usage with Vite or Vitest

This package is published as **ESM-only** and its built output contains `.svg` imports.
Because Vite's dependency pre-bundler (esbuild) does not handle `.svg` imports natively, add the following to your Vite or Vitest configuration:

```ts
// vite.config.ts
export default defineConfig({
  server: {
    deps: {
      inline: ['@pplancq/shelter-ui-react'],
    },
  },
});
```

```ts
// vitest.config.ts
export default defineConfig({
  test: {
    server: {
      deps: {
        inline: ['@pplancq/shelter-ui-react'],
      },
    },
  },
});
```
